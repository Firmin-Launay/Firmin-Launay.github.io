Python Divia API
“Python Divia API” is a simple Python library, which aims at providing a simple way to retreive “TOTEM” schedules of Divia (the city of Dijon’s public transit authority) bus and tramways, along with “DiviaVélodi” free-floating bikes availabilities. It is heavily inspired of its equivalent in JavaScript, developed by @gauthier-th.
