Générateur d’attestation de déplacement dérogatoire
“Générateur d’attesation de déplacement dérogatoire” is a tiny desktop app (Linux, macOS, Windows) I developed in Python, which allowed to generate a testimonial for the lockdown of October-November 2020, and then for the curfew until March 2021, in France. This app was only disponible in French. This project is now archived and abandoned.

RaspiMote
“RaspiMote” aims at turning a Raspberry Pi into a programmable keyboard / controller / remote control... It is developed by @A-delta and I, using multiple languages, such as Python, JavaScript, PowerShell or Batch, and it is compatible with both Linux and Windows. No release has been published for the moment but it is almost ready.

Stickopy
“Stickopy” is a little piece of software, coded in Dart and still in early stages of development, which has for objective to allow you to copy the whole content of a USB drive automatically (when plugged), quickly and quietly.
